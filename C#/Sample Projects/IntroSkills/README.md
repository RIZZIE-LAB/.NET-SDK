# IntroSkills

*This example was last tested on `robotVersion 1.7.4.9986`.*

The `IntroSkills` solution contains a sandbox of runnable C# skills for learning about Misty's .NET SDK. For more information about the code in this solution, see the guide in the [Misty Robotics developer documentation](https://docs.mistyrobotics.com/misty-ii/dotnet-sdk/sample-project/).
